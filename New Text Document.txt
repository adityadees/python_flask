For Linux and Mac:
$ export FLASK_APP=web
$ export FLASK_ENV=development
$ flask run

For Windows cmd, use set instead of export:
> set FLASK_APP=web
> set FLASK_ENV=development
> flask run

For Windows PowerShell, use $env: instead of export:
> $env:FLASK_APP = "web"
> $env:FLASK_ENV = "development"
> flask run
